## wars-an.py : 将棋ウォーズの棋譜を解析するためのプログラム

とりあえず棋譜収集の機能を作っていく

### 将棋ウォーズのrobots.txt
棋譜をクローリングするわけなので，一応robots.txtを確認しておこう，と思って見てみたら全文コメントアウトされていた．そのうち編集される可能性が高いので，プログラムを使用する際には注意．

[shogiwars.heroz.jp/robots.txt](http://shogiwars.heroz.jp/robots.txt)
